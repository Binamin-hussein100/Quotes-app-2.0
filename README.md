## PROJECT NAME
QUOTES ANGULAR

## DECRIPTION
A WEB A THAT ALLOWS ITS USERS TO READ QUOTES OR POST THEM <br>
IT HAS BEEN BUILT TO ENABLE USERS TO DELETE OR ADD TO THE QUOTES <br>
ALSO ALLOWS ITS USERS TO UP-VOTE OR DOWN-VOTE THE QUOTES.
